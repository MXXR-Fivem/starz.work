## ▶️ Run Locally

1. Clone the repo and go into the folder : 
```bash
git clone git@github.com:EpitechBachelorPromo2028/B-YEP-200-PAR-2-1-jobaggregator-14/.git
cd B-YEP-200-PAR-2-1-jobaggregator-14
```

2. Install docker

3. Install dependencies & start :
```bash
docker compose -f deploy/localhost/docker-compose.yml up --build 
```
