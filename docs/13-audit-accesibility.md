# Rapport d’accessibilité — STARZ

## Navigation et expérience utilisateur

Une attention particulière a été portée sur :

- la lisibilité des contenus ;
- la clarté des interfaces ;
- la simplicité du parcours utilisateur ;
- l’adaptation responsive mobile / desktop.


## Contrastes et lisibilité

Les couleurs et contrastes ont été pensés afin de conserver une bonne lisibilité des contenus sur l’ensemble du site.


## Accessibilité des composants

Les composants interactifs ont été développés avec une attention portée sur :

- les comportements clavier ;
- les états interactifs ;
- la cohérence visuelle ;
- les attributs d’accessibilité principaux.
